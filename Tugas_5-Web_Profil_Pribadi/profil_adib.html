WEBSITE PROFIL SEDERHANA

Ahmad Adib Adzka

Halo, saya Ahmad Adib Adzka, seorang pelajar SMK jurusan Rekayasa Perangkat Lunak (Software Engineering) yang bersemangat. Saat ini saya fokus mengembangkan keahlian dalam pengembangan web full-stack. Saya dalam proses menguasai beberapa bahasa pemrograman seperti Python, PHP, Java, JavaScript, HTML, dan CSS. Selain itu, saya juga memiliki minat di bidang Desain Grafis dan AI/ML.

Tujuan jangka panjang saya adalah menjadi Web Developer Full-stack profesional dan berkontribusi pada proyek-proyek inovatif. Saya percaya bahwa kemampuan saya dalam menyerap teori dan logika, dikombinasikan dengan kemauan untuk terus belajar dari yang lebih ahli, adalah aset utama saya. Beberapa soft skill yang sedang saya kembangkan meliputi manajemen waktu, kerja tim, pemecahan masalah (problem-solving), dan komunikasi efektif.

Sebagai bagian dari proses belajar, saya secara aktif mengerjakan berbagai proyek pribadi untuk mengaplikasikan ilmu yang saya dapatkan. Beberapa di antaranya termasuk:

  • 1. Website Profil Pribadi: Sebuah situs web sederhana yang menampilkan identitas dan proyek-proyek saya.
  • 2. AI Logika Sederhana: Proyek dasar yang mengeksplorasi konsep-konsep kecerdasan buatan.
  • 3. Website Data Perpustakaan: Sebuah situs web yang terintegrasi dengan database untuk mengelola data buku dan peminjam.

Melalui proyek-proyek ini, saya terus mengasah kemampuan teknis dan siap menghadapi tantangan di industri.
